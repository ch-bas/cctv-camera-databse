# Code of Conduct

## Our pledge

This project is a community-maintained open dataset. We welcome contributions from security professionals, installers, homelab enthusiasts, researchers, and anyone with camera knowledge to share.

We are committed to making participation a respectful experience for everyone, regardless of background, experience level, or technical expertise.

## Expected behaviour

- Be respectful and constructive in issues, pull requests, and discussions
- Assume good faith — a wrong spec is usually a mistake, not malice
- Keep discussions focused on the data and the project
- Welcome newcomers and explain conventions patiently

## Unacceptable behaviour

- Personal attacks, insults, or harassment of any kind
- Deliberate submission of false specifications to mislead users
- Spam, promotional content, or off-topic solicitation

## Scope

This Code of Conduct applies to all project spaces: GitHub issues, pull requests, discussions, and any other forum where project members interact.

## Reporting

If you experience or witness unacceptable behaviour, please open a private report via GitHub's [report abuse](https://docs.github.com/en/communities/maintaining-your-safety-on-github/reporting-abuse-or-spam) flow, or email the maintainer directly (address in the GitHub profile).

Reports will be reviewed promptly and handled confidentially.

## Enforcement

Maintainers reserve the right to remove comments, close issues, or ban contributors who violate this Code of Conduct.
